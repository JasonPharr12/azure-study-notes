Lab 01 - Secure Virtual Network Architecture
Date Completed: June 2026
Objective
Design and deploy a secure network environment in Azure using Virtual Networks, subnets, and Network Security Groups.
Resources Built

Resource Group: RG-Lab-01
Virtual Network: VNet-Lab-01 (10.0.0.0/16)
Subnet-Public: 10.0.0.0/24
Subnet-Private: 10.0.1.0/24
Network Security Group: NSG-Lab-01

Security Controls Implemented

NSG associated with both public and private subnets
Custom inbound rule allowing RDP on port 3389
All other inbound traffic denied by default

What I Learned

How to design a segmented network architecture in Azure
How Network Security Groups control traffic between subnets
How to apply least privilege principles to network security rules
